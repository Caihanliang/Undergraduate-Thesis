## 💡 **常见问题**

### **Q1: 我应该用 `.dat` 还是 `.npz`？**

**A:** 用 `.npz`！你的项目使用的是 `.npz` 格式（`his.npz`）。

- `.dat` 是 BasicTS 旧版本的格式
- `.npz` 是 NumPy 原生格式，更常用

---

### **Q2: his.npz 里面是什么结构？**

**A:** 标准的 NumPy 压缩数组：

```python
import numpy as np

# 加载
data = np.load('his.npz')

# 查看键名
print(data.files)  # ['data']

# 获取数组
array = data['data']
print(array.shape)  # [1416, 160, 3]
# T=1416 时间步
# N=160 站点
# C=3 特征 [流量, TOD, DOW]
```

---

### **Q3: 多变量数据集也是 his.npz 吗？**

**A:** 是的！格式完全一致，只是特征数不同：

```
单变量: his.npz → [1416, 160, 3]  (流量 + TOD + DOW)
多变量: his.npz → [1416, 160, 2]  (小客车 + 非小客车)
```

---
